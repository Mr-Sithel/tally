### Tally Info
- This was inspired by [AddonsXI's](https://github.com/AddonsXI) [Conquest](https://github.com/AddonsXI/conquest) addon.

### Overview
Displays FFXI's Conquest rankings, outpost regions, and various types of currencies. Vana'diel's balance of power is displayed from first to last with a displayed number to show how many regions each nation controls. In the currency tab, your nation is shown with an asterisks by it.

Currency updates upon logging in, zoning or has a manual update setting.

### Changed
- This has since changed a lot from the original Conquest addon with more features added.
- Now displaying in a ImGui window.

### Added
- Added: Regions tab that displays conquest control with outposts.
- Added: Areas tab for a quick search of areas across Vana'diel.
- Added: Currency tab for displaying conquest points, imperial standing, beastmen seal's, kindred seal's, and ancient beastcoins
- Added: 7 colored themes.

### Installation
* Download and unzip the correct version of Tally at https://github.com/Mr-Sithel/tally/releases/
* Copy the `tally` folder from inside of the `tally-main` folder into your Ashita addons directory
* Addon directory is : `HorizonXI\Game\addons`
* This was created for `Ashita (Interface v4.30)`, but `Ashita (Interface v4.16)` should still work which Horizon uses.
* You can load the addon by typing `/load addon tally`.  It is recommended you add this line to the appropriate place in `scripts/default.txt` to auto load.
