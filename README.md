### Tally Info
- This was forked from [AddonsXI's](https://github.com/AddonsXI) [Conquest](https://github.com/AddonsXI/conquest) addon.

### Overview
Displays FFXI's Conquest rankings and outpost regions in an order from 1st to last in a ImGui window.

### Changed
- I changed the name just to avoid confusion, because it displays a little differently.
- Displays in a ImGui window.

### Added
- With version 1.2.0+ the window theme and color was made to look like HXUI
- Region Area tab with search.
- Added outpost areas tied to the region.

### Installation

* Download and unzip the correct version of Tally at https://github.com/Mr-Sithel/tally/releases/
* Copy the `tally` folder from inside of the `tally-main` folder into your Ashita addons directory
* Addon directory is : `HorizonXI\Game\addons`
* This was created for `Ashita (Interface v4.30)`, but `Ashita (Interface v4.16)` should still work which Horizon uses.
* You can load the addon by typing `/load addon tally`.  It is recommended you add this line to the appropriate place in `scripts/default.txt` to auto load.
